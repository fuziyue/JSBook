跨域 ：
			        			        
			        同源策略 ： 同源策略（Same origin policy）是一种约定，它是浏览器最核心也最基本的安全功能，如果缺少了同源策略，则浏览器的正常功能可能都会受到影响。可以说Web是构建在同源策略基础之上的，浏览器只是针对同源策略的一种实现。
			        
			        
			                同源 ：域名，协议，端口。
			                
			                         域名 ：    小名ip   10.0.0.43  == www.biadu.com
			                         
			                         xiaomi.com   taobao.com
			                         
			                    dns解析器 : 
			                    
			                         协议  : http   https  file 
			                        
			                        
			                         端口 :   1-100000 :  8080
			                         http（协议）://127.0.0.1（IP）:8020（端口）/kuayu/%E8%B7%A8%E5%9F%9F.html?__hbt=1524206021094
			  
			  
			     页面和后台文件必须一致 ：才能使用ajax；
			  
			  		
			     在同源里面，只要有一个不满足，就是域名，协议，端口，我们就叫他跨域；

文件的类型分类 ： 
			         计算机是按文件里面的内容来划分
			         
			        
			        srcript  : src 
			        
			    jsonp :  属性src可以引进资源，如果这个资源长得想js
			    		,浏览器会把这个资源当成函数调用，
			    		
			    	原理：
			    		借用script标签加载资源没有跨域的限制，引进一个长得像js的资源，
			    		就是一个函数调用，这个函数里面有一个实参，实参就是我要用
			    		的数据，事先先写好一个全局函数，函数名要一致，
			    		直接在全局函数里面操作数据就行。
